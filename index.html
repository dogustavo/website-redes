Especialização em Desenvolvimento Web - Front-End

A disciplina aborda as condições básicas para o desenvolvimento Web, medianteacompreensão, domínio e aplicação das técnicas inerentes ao desenvolvimento deWebsites.Compreendendo o estudo do desenvolvimento de páginas Web dinâmicas através dasLinguagensHTML, CSS e JavaScript e o auxílio de bibliotecas e frameworks, propiciando ao estudanteumacompreensão global do funcionamento de um sistema Web

Objetivos

  • Criar e alterar Websites com a linguagem de marcação de hipertexto HTML.
  • Desenvolver padrões de apresentação com CSS.
  • Criar páginas otimizadas usando JavaScript e bibliotecas interativas.
  • Ser capaz de criar sites responsivos para dispositivos móveis utilizando frameworkscomoapoio ao desenvolvimento.

Conteúdo Programático

  • HTML: introdução, padrões, tags, formulários e seus componentes.
  • CSS: introdução, história, sintaxe geral,seletores, formatação, classes e pseudoclasses,propriedades de formatação, comportamento visual, o modelo de caixa e posicionamento.
  • JavaScript: Introdução, histórico, características, estruturas condicionais, estruturasderepetição, vetores, funções, objetos, manipulação de HTML.
  • Frameworks Front-End: introdução, principais frameworks, características, utilização, Web.
